# SITI 補助計畫：團隊核心成員學經歷亮點 (草案)

本文件旨在為 SITI 補助計畫提供團隊成員的「實績與亮點」描述，將 DropOut 團隊包裝為具備高技術執行力與商業洞察力的 B2B AI 解決方案團隊。

---

## 1. 孟一 (DreamOne) - 計畫主持人 / 營運總監
*   **定位**: 數位轉型策略專家與高層決策者。
*   **亮點**:
    *   具備豐富的專案監督與架構性決策經驗，擅長於複雜商業環境中識別技術落地路徑。
    *   在 DropOut 擔任執行長，負責永齡、鄉育等大型教育服務專案之資源對接與流程規範定調。
    *   高度重視資料合規性與系統化紀錄，確保企業級軟體交付之嚴謹性。

## 2. 嘉尹 (David) - 技術總監 / 系統架構師
*   **定位**: 連結技術深度與商業需求的雙棲專家。
*   **亮點**:
    *   具備 Headless 商務架構 (Medusa) 與主流平台 (Shopify/Wix) 之深度對比評估與實作經驗。
    *   擅長處理 B2B 複雜定價邏輯與跨系統 API 串接 (如 Notion, Line, 支付金流)。
    *   於「富據智能」等高活躍專案中擔任主要技術協調者，確保開發進度與客戶需求精準對齊。

## 3. 宜津 (Yijin) - 產品經理 / 流程分析師
*   **定位**: 企業 SOP 邏輯梳理與 AI 功能定義的核心。
*   **亮點**:
    *   擅長將企業模糊的業務需求轉化為可執行的技術開發清單 (Requirement Engineering)。
    *   主導企業內部 SOP 的數位化重構，負責產品功能驗證與用戶體驗 (UX) 優化。
    *   在多專案環境下具備卓越的協作時間調度能力，確保多方溝通之透明度與效率。

## 4. 懶懶企鵝博士 - AI 研發工程師 / 核心開發者
*   **定位**: AI 模型落地與自動化流程實作的技術引擎。
*   **亮點**:
    *   專精於 AI Agent 開發、模型微調 (Fine-tuning) 與地端/雲端中繼站 (Relay Server) 之佈署。
    *   具備快速解決技術瓶頸的實作能力，曾於「Lets-Play」等技術高壓專案中擔任核心負責人。
    *   擅長利用自動化腳本優化內部工作流，將最新 AI 研究轉化為具備生產力之工具。
